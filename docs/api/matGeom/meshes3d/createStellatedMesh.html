createStellatedMesh

PURPOSE

CREATESTELLATEDMESH Replaces each face of a mesh by a pyramid.

SYNOPSIS

function varargout = createStellatedMesh(vertices, faces, varargin)

DESCRIPTION 

CREATESTELLATEDMESH  Replaces each face of a mesh by a pyramid.

   [V2, F2] = createStellatedMesh(V, F)

   Example
     [v, f] = createCube
     [v2, f2] = createStellatedMesh(v, f);
     figure; drawMesh(v2, f2); axis equal; view(3);

   See also
     meshes3d, drawMesh
