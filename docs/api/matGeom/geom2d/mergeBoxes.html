mergeBoxes

PURPOSE

MERGEBOXES Merge two boxes, by computing their greatest extent.

SYNOPSIS

function box = mergeBoxes(box1, box2)

DESCRIPTION 

MERGEBOXES Merge two boxes, by computing their greatest extent.

   BOX = mergeBoxes(BOX1, BOX2);

   Example
   box1 = [5 20 5 30];
   box2 = [0 15 0 15];
   mergeBoxes(box1, box2)
   ans = 
       0 20 0 30


   See also
   boxes2d, drawBox, intersectBoxes
