clipPoints

PURPOSE

CLIPPOINTS Clip a set of points by a box.

SYNOPSIS

function points = clipPoints(points, box)

DESCRIPTION 

CLIPPOINTS Clip a set of points by a box.

   CLIP = clipPoints(POINTS, BOX);
   Returns the set of points which are located inside of the box BOX.


   See also
