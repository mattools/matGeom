edgeLength3d

PURPOSE

EDGELENGTH3D Return the length of a 3D edge.

SYNOPSIS

function len = edgeLength3d(edge, varargin)

DESCRIPTION 

EDGELENGTH3D Return the length of a 3D edge.

   L = edgeLength3D(EDGE);  
   Returns the length of a 3D edge, with following representation:
   [x1 y1 z1 x2 y2 z2].

   Example
     p1 = [1 1 1];
     p2 = [3 4 5];
     edge = createEdge3d(p1, p2);
     edgeLength3d(edge)
     ans =
         5.3852
   
   See also
     edges3d, createEdge3d, drawEdge3d
