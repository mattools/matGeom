mergeBoxes3d

PURPOSE

MERGEBOXES3D Merge 3D boxes, by computing their greatest extent.

SYNOPSIS

function box = mergeBoxes3d(box1, box2)

DESCRIPTION 

MERGEBOXES3D Merge 3D boxes, by computing their greatest extent.

   BOX = mergeBoxes3d(BOX1, BOX2);

   Example
   box1 = [5 20 5 30 10 50];
   box2 = [0 15 0 15 0 20];
   mergeBoxes3d(box1, box2)
   ans = 
       0 20 0 30 0 50


   See also
   boxes3d, drawBox3d, intersectBoxes3d
