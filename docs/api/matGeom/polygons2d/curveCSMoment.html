curveCSMoment

PURPOSE

CURVECSMOMENT Compute centered scaled moment of a 2D curve.

SYNOPSIS

function m = curveCSMoment(curve, p, q)

DESCRIPTION 

CURVECSMOMENT  Compute centered scaled moment of a 2D curve.
   M = curveCSMoment(CURVE, P, Q)
