knnGraph

PURPOSE

KNNGRAPH Create the k-nearest neighbors graph of a set of points.

SYNOPSIS

function edges = knnGraph(nodes, varargin)

DESCRIPTION 

KNNGRAPH Create the k-nearest neighbors graph of a set of points.

   EDGES = knnGraph(NODES)

   Example

   edges = knnGraph(nodes);
